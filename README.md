# cherry-parent

```
├── cherry-parent
│   ├── cherry-common  # 存放共用的东西(常用工具包等)
│   ├── cherry-member  # 会员域
│   │   ├── cherry-member-api  # 会员域开放的api接口
│   │   ├── cherry-member-biz  # 会员域接口的实现
│   ├── cherry-biz  # 具体的业务处理（调用各个域进行业务处理）
│   ├── cherry-web  # 后台web项目
│   ├── cherry-job  # job作业
```

## 项目使用技术
* jdk 1.7
* jetty9
* struts2
* spring3
* ibatis
* cat监控
* sql server
